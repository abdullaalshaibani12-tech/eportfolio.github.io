Unit 2: Data Structures and Basic Operations in R

Unit 2 focused on learning about different types of data structures in R and how to effectively perform operations using them. I explored the fundamental data structures such as vectors, matrices, lists, and data frames, which form the backbone of data manipulation in R. The unit emphasized performing and executing various R commands to manage and analyze data efficiently. Additionally, I developed an understanding of vectors and matrices, including how to perform basic arithmetic operations using different operators in R. These skills provided a strong foundation for handling data and performing analytical computations.

Key Learnings:

  • Understanding different types of data structures in R.
  • Performing and executing R commands effectively.
  • Exploring vectors and matrices as essential components of data analysis.
  • Performing basic arithmetic operations using various operators in R.
  • Applying arithmetic operations to vectors and matrices for analytical purposes.
